# Lessons Learned

This project provided practical hands-on experience in deploying and administering a Windows Server environment.

## Skills Acquired

During this project, I gained practical experience in:

- Installing Windows Server 2025
- Deploying Active Directory Domain Services (AD DS)
- Configuring DNS and DHCP
- Creating Organizational Units (OUs)
- Managing users and security groups
- Configuring NTFS permissions
- Joining Windows clients to an Active Directory domain
- Creating and applying Group Policy Objects (GPOs)
- Troubleshooting common Active Directory issues

## Challenges

Throughout the project, several technical challenges were encountered and resolved, including:

- Joining a Windows 11 client to the domain
- Configuring DNS name resolution
- Deploying and validating DHCP services
- Troubleshooting NTFS permission inheritance
- Applying and validating Group Policy settings

## Conclusion

This project successfully simulated a small business Active Directory environment and significantly strengthened my practical Windows Server administration skills.

It also improved my understanding of enterprise network services, user management, security permissions, and Group Policy administration.
